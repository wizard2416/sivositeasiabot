طريقة الاستخدام

1. Submit a card

POST /api/external/submit-card
Headers: X-API-Key: your_api_key
Body: {
    "pin": "1234567890123456",
    "external_ref": "your-reference-id",
    "webhook_url": "https://your-site.com/webhook"
}
Response: {
    "success": true,
    "card_id": 123,
    "status": "pending"
}

2. Submit multiple cards

POST /api/external/submit-cards
Headers: X-API-Key: your_api_key
Body: {
    "cards": [
        {"pin": "1234567890123456", "external_ref": "ref-1"},
        {"pin": "9876543210987654", "external_ref": "ref-2"}
    ],
    "webhook_url": "https://your-site.com/webhook"
}

3. Check card status

GET /api/external/card-status/{card_id}
Headers: X-API-Key: your_api_key
Response: {
    "card_id": 123,
    "status": "verified",
    "amount": 5000,
    "result_message": "..."
}

4. Webhook callback

// Your webhook receives POST:
{
    "card_id": 123,
    "external_ref": "your-reference-id",
    "status": "verified" | "failed",
    "amount": 5000,
    "result_message": "..."
}

5. List all cards

GET /api/external/cards?status=pending&limit=50
Headers: X-API-Key: your_api_key
